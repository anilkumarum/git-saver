Thanks for installing gitSaver

This guide help you to understand how codeBook work and How to use gitSaver?

Eable/disable gitSaver

gitSaver is disable by default. Click on extension icon to enable gitSaver

Select files and folders

Hover over file or folder name, checkbox appear. Check the checkbox to select file or folder. SaveBtn will apear Check the first checkbox to toggle markall/unmarkAll

Save selected files and folders into local folder

Select folder or edit destination folder path. Click on Add folder to add new folder. Click on Save Files button to save selected files and folders into selected folder.
